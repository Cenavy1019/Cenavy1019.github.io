博客搭建

第一步 下载nodeJS

1.安装nodeJS：https://nodejs.org/en/
2.测试nodeJS    终端里输入node -v

第二步 安装hexo

1.运行npm install hexo -g 命令
2.运行root Artasdlkajs 命令
3.在sudo nam install hexo -g 然后输入自己的密码
4.测试hexo
5.创建目录进入，hexo init 是初始化hexo目录，hexo server 启动本地服务

第三步 github账号

一、配置sshkey
1.终端命令ssh-key gen 按三次回车
2.vim ~/.ssh/id_rsa.pub 复制ssh有用
3.github 设置→ssh栏里 粘贴刚才复制的ssh

二、配置github与本地机器验证
1.git config —global user name “请输入自己github的用户名” 
2.git config —global user name “请输入自己的github密码”

三、新建博客域名仓库
用户名.github.io

第四步 修改config.yml文件

1.找到deploy字段
2.在type后面加上 （要加空格，请去掉括号）git
3.repo： 复制粘贴域名仓库的http地址

第五步 发布自己的文档

1.nam install heao-deployer-git—save
2.终端输入hexo g 进行文件转html和css及js格式文件
3.终端输入hexo d 进行上传，看到done：git说明已经成功，打开自己的博客域名地址就能看到了。

PS 修改主题

1.找到自己喜欢的主题的域名，复制
2.终端里先进入主题目录
3.输入git clone http：//我是主题域名地址, 回车 下载主题
4.打开config.yml文件，找到themes，修改成新主题名字
5.跳转到blog目录下，输入hexo server启动服务
6.复制网址查看是否变动。
7.还可以自己修改背景图。
